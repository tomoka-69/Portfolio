top page

top pageでは、
ライブラリー、検索、
メニューバーを
設置することで、
機能を分けている。

library page

library pageは、
検索ボックスがあり、
部分入力でも
検索が可能。

書籍名を
ダブルクリック
することで、
書籍情報を
確認することが
できる。

書籍名を
クリック後、
貸出ボタンで貸出、
返却ボタンで
返却できる。

search page

　tkinterの基本的な使い方を学ぶために作成。シンプルなアプリだが、ボタンや画像の配置などを工夫して見やすさを意識した。画像にはないものの、登録画面とログイン画面も作成。もし次に作ることがあれば、改善したい点や追加したい機能が多くある。
